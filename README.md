## Getting Started (Local Setup)

### Prerequisites
- Node.js **18.x**
- npm

### Installation

```bash
git clone https://github.com/your-username/personal-portfolio.git
cd personal-portfolio
npm install
```

### Run locally
```bash
npm start
```

The app will run on:  
http://localhost:3000

---

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build/` folder.
